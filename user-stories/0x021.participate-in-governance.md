# Participate In Governance

As a user, I want to participate in governance so I can help decide future fee
splits or how buybacks are executed.

## Acceptance Criteria

- [ ] Eligible token holders can view active proposals affecting fee splits and
      buyback execution parameters.
- [ ] A holder can cast a vote from the UI using their connected wallet, with
      voting weight derived from their on-chain holdings at the proposal
      snapshot.
- [ ] Vote tallies and proposal outcomes are visible in real time and link to
      the on-chain governance program.
- [ ] Passed proposals that change fee splits or buyback parameters take effect
      automatically and the new values are reflected in the fee breakdown UI.

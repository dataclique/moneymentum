# Earn Bounties For Contributions

As a user, I want to earn bounties for contributing to protocol growth, so I'm
incentivized to take part in tasks.

## Acceptance Criteria

- [ ] A list of open bounties is visible in the UI with task description, reward
      amount, and acceptance criteria.
- [ ] A user can claim a bounty, submit proof of completion, and track the
      review status of their submission.
- [ ] Approved submissions trigger an on-chain payout to the contributor's
      wallet, with the transaction linked from the UI.
- [ ] Bounty funding draws from the configured fee allocation and the remaining
      bounty budget is visible to users.

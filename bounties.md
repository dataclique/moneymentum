# Bounties

Moneymentum offers bounties for valuable contributions to encourage community participation and reward quality work.

## How Bounties Work

- Issues labeled with `bounty` have associated rewards
- Bounty amounts are specified in issue descriptions or comments
- Contributors earn bounties by successfully completing bounty-labeled issues
- Payments are processed after PR merge and verification

## Bounty Types

### Bug Fixes
- Critical bugs: $100-500
- Major bugs: $50-200  
- Minor bugs: $10-50

### Feature Implementation
- Major features: $200-1000
- Minor features: $50-300
- UI/UX improvements: $25-150

### Documentation
- Technical documentation: $25-100
- User guides: $15-75
- Code comments/examples: $10-50

## Claiming Bounties

1. Comment on the bounty issue to claim it
2. Submit a PR that addresses all acceptance criteria
3. Pass code review and merge
4. Bounty is awarded after verification

## Payment Methods

- GitHub Sponsors
- Cryptocurrency (preferred)
- PayPal (for larger amounts)

## Rules

- Only one bounty per issue
- First valid submission wins
- Must follow contribution guidelines in [contributions.md](./contributions.md)
- Bounties expire after 30 days if unclaimed
- Maintainers reserve right to adjust bounty amounts
